# Floating Orbs

Floating Orbs is a RuneLite plugin that adds movable prayer and special attack orbs.

## Features

- Movable prayer orb
- Movable special attack orb
- Optional snap mode to keep special next to prayer
- Custom colors, size, blink settings, and points display
- Round and square orb modes

## Development

Requirements:

- Java 17
- Gradle (wrapper included)

Build:

```bash
./gradlew build
```

Run test launcher:

```bash
./gradlew testClasses
```

Then run `FloatingOrbsPluginLauncher` from your IDE.

## Notes

- This plugin is designed for 1-click interactions only.
- Keep usage compliant with Jagex and RuneLite rules.
